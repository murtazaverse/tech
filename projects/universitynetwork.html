University Network System using CISCO

  • Tech Stack: CISCO Packet Tracer, Network Programming
  • Github URL: Project Link

The project involves designing a star-topology network using Cisco Packet Tracer for a university. It enables effective resource sharing, communication, file sharing, and result publishing within the university.

The network implements access control measures to ensure security. Specific rooms and PCs have restricted access, while others have limited connectivity based on their roles and functions.

The project considers the connectivity rules between different rooms within the university, promoting efficient communication and collaboration. The network is divided into floors, with specific access permissions between rooms on the same floor and between different floors.
